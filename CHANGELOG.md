# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-12-22

### Added
- Initial release
- `AudioColumn` for Filament Tables
- `AudioEntry` for Filament Infolists
- `AudioField` for Filament Forms
- Circular progress indicator with SVG
- Play/Pause functionality
- Auto-pause when multiple players (only one plays at a time)
- Volume control (optional)
- Duration display (optional)
- Customizable size and progress color
- Dark mode support
- Support for local and remote audio URLs
